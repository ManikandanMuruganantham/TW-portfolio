## Introduction
LightWorks is a free video-editing software with advanced editing and mastering capabilities. This guide explains essential tasks:
- Install and activate LightWorks  
- Import and trim video and audio  
- Edit media  
- Export content  

## Installation and Activation

### Creating your account
1. Visit the LightWorks website  
2. Register an account  
3. Enter personal details and verify email  
4. Follow activation steps  

> Note: Install the latest QuickTime version before starting

### Windows installation guide
1. Go to Downloads → Windows  
2. Download the appropriate version (32-bit or 64-bit)  
3. Run the `.exe` file  
4. Complete the installation wizard  
   - Select all components  
   - Assign a machine number  
5. Restart the system  
6. Launch and activate LightWorks  

### Mac installation guide
1. Download the `.dmg` file  
2. Open the file  
3. Drag LightWorks to Applications  
4. Launch and activate  

### Linux installation guide
1. Download `.deb` or `.rpm`  
2. Open and install  
3. Launch and activate  

## Basic Editing
Create a project, import media, edit clips, and export content

## Importing Content
1. Select **Create a new project**  
2. Enter project details  
   - Name  
   - Frame rate  
3. Open **Local Files**  
4. Select files  
   - Use Shift or Ctrl for multiple selection  
5. Click **Import**  

## Cutting and Editing Clips
1. Open the **Edit** tab  
2. Select a clip and double-click  
3. Use the timeline and markers to define clip sections  
4. Insert clips into the sequence  

## Trimming
1. Adjust cut points in the timeline  
2. Drag edges of clips to trim  

## Adding Audio
1. Select audio from project panel  
2. Open in source viewer  
3. Mark start and end points  
4. Insert into timeline  

## Editing Audio
1. Enable key-framing using Shift  
2. Adjust volume using track line  
3. Drag to increase or decrease audio  
4. Add fade-in or fade-out effects  

## Transitions
1. Right-click between clips  
2. Select **Transitions**  
3. Choose type and duration  

## Video Effects
Available effects include:
- 3D DVE  
- Color correction  
- Titles  

## 3D DVE (Digital Video Effects)
Use controls to adjust:
- Rotation  
- Position  
- Scale  
- Crop  
- Shadow  

## Color Correction
Adjust image colors using:
- Balance  
- RGB  
- HSV  
- Curves  

Control:
- Shadows  
- Midtones  
- Highlights  
- Saturation  

## Titles
Add titles using:
- Static  
- Crawling  
- Rolling  

Steps:
1. Position timeline marker  
2. Enter text  
3. Adjust font and effects  

## Exporting Content
1. Right-click in sequence viewer  
2. Select **Export**  
3. Choose format  
   - Blu-ray  
   - DVD  
   - Vimeo  
   - YouTube  
4. Adjust settings  
   - Frame rate  
   - Size  
5. Click **Start**
